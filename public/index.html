Local Security Shield

FormSarthi operates as a zero-knowledge local-first vault. Here's how your data stays safe.

🛡️

Zero Server-Side Storage

Your documents, identity details, bank specifications, and login PIN never leave this device. The backend server acts only as a stateless utility engine for running OCR processing and converting image configurations.

🔒 AES-256 Bit Encryption

All database elements are encrypted in your browser's private IndexedDB storage using AES-256 standard encryption. Decryption only takes place in active system memory when successfully unlocked.

🔑 PBKDF2 Key Derivation

Decryption keys are generated dynamically inside the browser client using PBKDF2 with 1,000 hash iterations. The key is derived directly from your unique email address salt and your private 6-digit PIN.

🧠 Stateless Session Access

When you unlock your vault, a decrypted copy is held briefly inside window sessionStorage. The Chrome Extension reads from this local memory tab-to-tab, completely bypassing API networks.

🚫 Zero Data Profiling

No analytics engines, trackers, or databases monitor your filled fields. FormSarthi is fully private and run locally by you, keeping your documents and metadata 100% isolated.

About FormSarthi

FormSarthi is your private, stateless utility engine for form filling. Designed for Indian students and citizens, it seamlessly processes and encrypts your data locally, ensuring your PAN, Aadhaar, and other sensitive details never leave your device unencrypted. We utilize on-device OCR and document analysis to make filling out tedious online applications a breeze.
